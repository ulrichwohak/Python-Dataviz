# Assignments

Homework delivery is a **per-term decision** and is intentionally **not** built into this
prototype repository.

When the instructor chooses a format (for example: short in-repo practice notebooks, a problem
set, or stand-alone debugging repositories), assignments will be added here or linked from
[`schedule/session_plan.md`](../schedule/session_plan.md).

For now, students should use:

- the per-week practice notebooks in [`notebooks/exercises/`](../notebooks/exercises/), and
- the final project scaffold in [`project_template/`](../project_template/).
