# Course Outline — Data Visualisation with Python

MSc Business Analytics, CEU. Eight weekly sessions, one notebook each, taught through live
coding. Every session uses synthetic data generated by
[`scripts/fetch_data.py`](../scripts/fetch_data.py), which performs no network downloads in
this prototype.

Each completed notebook follows the same shape: a motivating business question, learning
objectives, data loading, a concept explanation, plots built step by step with written
interpretation, "pause and predict" prompts, common mistakes, one AI co-pilot activity, a
verification checklist for AI output, exercises, and at least one figure saved to `figures/`.

| Week | Topic | Datasets | Core skills |
| --- | --- | --- | --- |
| 1 | Intro to visualisation | sales | Why visualise; the anatomy of a chart; first matplotlib plots; chart choice |
| 2 | Data for visualisation | transactions, customers | Tidy data; reshape with pandas; aggregation; handling missing values for plots |
| 3 | Core business charts | sales | Line, bar, grouped/stacked bar; time series; small multiples |
| 4 | Distributions and comparisons | operations, customers | Histograms, boxplots, violin plots; group comparison; outliers and skew |
| 5 | Visual design and storytelling | sales, customers | Colour, annotation, decluttering, narrative; redesigning a weak chart |
| 6 | Geospatial visualisation | regions, stores | Choropleths and point maps with geopandas; CRS basics; classification |
| 7 | Interactive visualisation and dashboards | dashboard | plotly interactivity; from notebook to a Streamlit dashboard |
| 8 | Project workshop and peer review | (student data) | Applying the workflow; critique; verification checklists |
